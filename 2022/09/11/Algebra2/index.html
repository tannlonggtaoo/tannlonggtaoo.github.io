NBG Set Theory

von Neumann–Bernays–Gödel集合论公理体系是ZFC公理体系的一个保守拓展（conservative extension）。NBG体系的一个特点是可以区分类（class）和集合（set）。类的概念比集合更为广泛，例如可以定义“所有集合所组成的类”。（根据正则公理，ZFC公理系统中不存在“所有集合所组成的集合”）。

事实上，保守拓展就是说：NBG可以谈论比集合更宽广的概念“类”，且关于集合的陈述在NBG中可证明当且仅当其在ZFC中可证明。

NBG的原始对象是类、一个表示“是集合”的谓词RRR和成员关系∈\in∈。

RRR的定义是：类xxx是集合（即R(x)R(x)R(x)）当且仅当存在一个类XXX，使得x∈Xx\in Xx∈X。不是集合的类称为真类（proper class）

（这种定义下）∈\in∈可以用在类与类之间。

注：这里有两种定义途径，以上显示的是Godel的叙述方法。另一种Bernays叙述方法则∈\in∈左侧只能是集合，但可以通过表示（representation）概念“圆”回去。

自同态即形同HomC(A,A)Hom_\mathrm{C}(A,A)HomC​(A,A)（记为EndC(A)End_\mathrm{C}(A)EndC​(A)）的态射集中的态射。

commutative diagram

若图上任意两个对象间的任意路径的合成都相等，则这个图是可交换的。

pointed set （category）

令C=SET\mathrm{C}=\mathbb{SET}C=SET，A={∗}A=\{*\}A={∗}（即某个固定的单元素集），则有点集范畴（记为SET∗\mathbb{SET}^*SET∗）定义为共切片范畴A/C={∗}/SETA/\mathrm{C}=\{*\}/\mathbb{SET}A/C={∗}/SET。

有点集范畴的对象就是函数f:{∗}→Sf:\{*\}\rightarrow Sf:{∗}→S（注意SET\mathbb{SET}SET的态射就是集合上定义的函数），其中SSS是集合。因此f(∗)∈Sf(*)\in Sf(∗)∈S，即有点集范畴的每个对象可以从每个集合中选择一个元素。因此，对象f:{∗}→Sf:\{*\}\rightarrow Sf:{∗}→S也可以用一个二元组等价地表示为(S,s)(S,s)(S,s)，其中SSS是fff的值域，s=f(∗)∈Ss=f(*)\in Ss=f(∗)∈S。

因此，有点集范畴的态射(S,s)→(T,t)(S,s)\rightarrow (T,t)(S,s)→(T,t)（可以等价地）指某个集合上的函数σ:S→T\sigma :S\rightarrow Tσ:S→T，且σ(s)=t\sigma (s)=tσ(s)=t。

有点集范畴的对象集Obj({∗}/SET)Obj(\{*\}/\mathbb{SET})Obj({∗}/SET)中的各个对象（形同(S,s)(S,s)(S,s)）就是有点集（pointed set，或based set，rooted set）。

有点集和普通集合的区别在于，每个有点集都存在一个被选出的、特殊的、可辨的特殊元。例如，这个特殊元在群论中就是指单位元。

subobject classifier

有时可以像讨论集合的子集一样讨论对象（未必是集合）的子对象（subobject）。对范畴C\mathrm{C}C和其中一个对象AAA而言，AAA的子对象和态射A→ΩA\rightarrow \OmegaA→Ω之间一一对应，其中Ω\OmegaΩ是C\mathrm{C}C的一个固定的对象。这个Ω\OmegaΩ就是子对象分类器。
